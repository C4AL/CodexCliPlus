# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和负责人，我们承诺让每个人都能在没有骚扰的环境中参与社区，不因年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教、性身份或性取向而受到区别对待。

我们承诺以有助于建立开放、友善、多元、包容和健康社区的方式行动与互动。

## 我们的标准

有助于营造积极社区环境的行为包括：

* 对他人表现出同理心与友善
* 尊重不同的意见、观点和经历
* 提出并坦然接受建设性反馈
* 为自己的错误承担责任，向受影响的人道歉，并从经历中学习
* 不只关注个人利益，也关注整个社区的最佳利益

不可接受的行为包括：

* 使用带有性意味的语言或图像，以及任何形式的性关注或性挑逗
* 恶意挑衅、侮辱性或贬损性评论，以及人身攻击或政治攻击
* 公开或私下骚扰他人
* 未经明确许可发布他人的私人信息，例如实体地址或电子邮件地址
* 其他任何在专业环境中可被合理认定为不当的行为

## 执行责任

社区负责人负责解释并执行可接受行为标准，并会针对其认为不当、威胁性、冒犯性或有害的行为采取适当且公平的纠正措施。

社区负责人有权也有责任移除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、议题和其他贡献，并会在适当时说明处理决定的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中正式代表社区的场景。代表社区的示例包括使用官方电子邮件地址、通过官方社交媒体账号发布内容，或作为指定代表参加线上或线下活动。

## 执行方式

可将滥用、骚扰或其他不可接受行为的事件报告给负责执行本准则的社区负责人，联系方式为 QQ 群。

所有投诉都会被及时、公正地审查和调查。

所有社区负责人都有义务尊重事件报告人的隐私与安全。

## 执行指南

社区负责人会依据以下社区影响指南，判断违反本行为准则的行为应承担的后果：

### 1. 纠正

**社区影响**：使用不当语言，或出现其他被认为不专业、不受社区欢迎的行为。

**处理后果**：社区负责人会发出私下书面警告，说明违规性质并解释该行为为何不当。可能会要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为构成违规。

**处理后果**：发出警告，并说明继续违规将产生的后果。在指定期限内，不得与相关人员互动，包括不得主动联系负责执行本行为准则的人员。这包括避免在社区空间以及社交媒体等外部渠道中互动。违反这些要求可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**处理后果**：在指定期限内临时禁止与社区进行任何形式的互动或公开交流。在此期间，不得与相关人员进行公开或私下互动，包括不得主动联系负责执行本行为准则的人员。违反这些要求可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出反复违反社区标准的行为模式，包括持续的不当行为、骚扰个人，或攻击、贬低某类群体。

**处理后果**：永久禁止在社区内进行任何形式的公开互动。
